# Twitter-clone

This is a very simple clone of Twitter UI built with TailwindCSS & VueJs.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
